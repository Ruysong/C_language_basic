# Week 03 - 데이터와 함수

## 학습 주제

- 함수 기본 구조, 매개변수와 반환값

## 문제 목록

### problem_01_circle_area_no_parameters - 원의 넓이 출력 - 매개변수와 반환값 없음

- 파일: `problem_01_circle_area_no_parameters.c`
- 요구사항:
  - 매개변수와 반환값이 없는 함수를 작성한다.
  - 반지름 1과 10인 원의 넓이를 각각 출력한다.
  - 함수 호출 흐름과 상수 정의를 연습한다.

### problem_02_circle_area_with_parameter - 원의 넓이 출력 - 매개변수 사용

- 파일: `problem_02_circle_area_with_parameter.c`
- 요구사항:
  - 반지름을 매개변수로 전달받는 함수를 작성한다.
  - 전달받은 반지름으로 원의 넓이를 계산해 출력한다.
  - 매개변수를 이용한 함수 재사용을 연습한다.

### problem_03_circle_area_return_value - 원의 넓이 계산 - 반환값 사용

- 파일: `problem_03_circle_area_return_value.c`
- 요구사항:
  - 반지름을 전달받아 원의 넓이를 반환하는 함수를 작성한다.
  - main 함수에서 반환값을 받아 출력한다.
  - 반환값이 있는 함수 구조를 연습한다.

## 실행 방법

```bash
gcc problem_01_circle_area_no_parameters.c -o problem_01_circle_area_no_parameters
./problem_01_circle_area_no_parameters
```
