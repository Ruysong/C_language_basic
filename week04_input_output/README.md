# Week 04 - 입출력문의 활용

## 학습 주제

- scanf/printf 기반 입출력 함수

## 문제 목록

### problem_01_basic_input_output - 사용자 입출력 연습

- 파일: `problem_01_basic_input_output.c`
- 요구사항:
  - 실수와 문자를 입력받는 함수를 작성한다.
  - 입력받은 값을 main 함수에서 확인해 출력한다.
  - 기본 입력 함수와 출력 형식을 연습한다.

### problem_02_cylinder_volume - 원기둥 부피 구하기

- 파일: `problem_02_cylinder_volume.c`
- 요구사항:
  - 반지름과 높이를 입력받는다.
  - 원기둥의 부피를 계산하는 함수를 작성한다.
  - 입력, 계산, 출력 역할을 함수로 나누어 구현한다.

## 실행 방법

```bash
gcc problem_01_basic_input_output.c -o problem_01_basic_input_output
./problem_01_basic_input_output
```
