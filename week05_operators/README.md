# Week 05 - 기본 연산자

## 학습 주제

- 산술 연산, 형 변환, 단위 변환

## 문제 목록

### problem_01_split_real_number - 실수의 정수부와 소수부 분리

- 파일: `problem_01_split_real_number.c`
- 요구사항:
  - 실수를 입력받아 정수부와 소수부로 분리한다.
  - 형 변환과 산술 연산을 사용한다.
  - 분리된 값을 보기 좋게 출력한다.

### problem_02_temperature_conversion - 온도 변환

- 파일: `problem_02_temperature_conversion.c`
- 요구사항:
  - 온도 값을 입력받는다.
  - 변환 공식을 적용해 다른 단위의 온도로 바꾼다.
  - 실수 연산과 함수 반환을 연습한다.

### problem_03_fuel_efficiency - 연비 계산

- 파일: `problem_03_fuel_efficiency.c`
- 요구사항:
  - 출발/도착 거리계 값과 사용 연료량을 입력받는다.
  - 주행거리와 연비를 계산한다.
  - 계산 결과를 함수 반환값으로 처리한다.

### problem_04_age_calculation - 나이 계산

- 파일: `problem_04_age_calculation.c`
- 요구사항:
  - 나이를 입력받는다.
  - 다음 해 또는 특정 기준의 나이를 계산해 출력한다.
  - 함수를 나누어 간단한 계산 흐름을 구현한다.

### problem_05_time_conversion - 초 단위 시간 변환

- 파일: `problem_05_time_conversion.c`
- 요구사항:
  - 초 단위 시간을 입력받는다.
  - 시/분/초 형태로 변환한다.
  - 나눗셈과 나머지 연산을 활용한다.

## 실행 방법

```bash
gcc problem_01_split_real_number.c -o problem_01_split_real_number
./problem_01_split_real_number
```
