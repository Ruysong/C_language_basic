# Week 09 - 배열 기본

## 학습 주제

- 배열 선언, 순회, 검색

## 문제 목록

### problem_01_print_array_elements - 배열 요소 출력

- 파일: `problem_01_print_array_elements.c`
- 요구사항:
  - 정수 배열을 선언하고 초기화한다.
  - 반복문으로 배열 요소를 순서대로 출력한다.
  - 배열 인덱스 접근을 연습한다.

### problem_02_find_minimum_value - 배열의 최솟값과 인덱스 찾기

- 파일: `problem_02_find_minimum_value.c`
- 요구사항:
  - 실수형 배열을 선언한다.
  - 반복문으로 최솟값과 첨자를 찾는다.
  - 배열 순회와 비교 연산을 연습한다.

### problem_03_input_integer_array - 정수 배열 입력과 출력

- 파일: `problem_03_input_integer_array.c`
- 요구사항:
  - 정수 배열에 값을 입력받는다.
  - 입력된 값을 순서대로 출력한다.
  - 배열 크기와 반복 입력을 다룬다.

### problem_04_search_array_value - 배열에서 특정값 검색

- 파일: `problem_04_search_array_value.c`
- 요구사항:
  - 정수 배열을 선언하고 검색할 값을 입력받는다.
  - 배열 안에 값이 존재하는지 확인한다.
  - 검색 결과와 인덱스를 출력한다.

## 실행 방법

```bash
gcc problem_01_print_array_elements.c -o problem_01_print_array_elements
./problem_01_print_array_elements
```
