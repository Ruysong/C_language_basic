# Week 11 - 문자열

## 학습 주제

- 문자 배열과 문자열 처리 함수 구현

## 문제 목록

### problem_01_string_length - 문자열 길이 계산

- 파일: `problem_01_string_length.c`
- 요구사항:
  - 문자열을 입력받는다.
  - 직접 구현한 함수로 문자열 길이를 계산한다.
  - 널 문자까지의 문자 개수를 반환한다.

### problem_02_string_copy - 문자열 복사

- 파일: `problem_02_string_copy.c`
- 요구사항:
  - 원본 문자열을 입력받는다.
  - 직접 구현한 함수로 대상 배열에 복사한다.
  - 복사된 문자열을 출력한다.

### problem_03_string_concat - 문자열 연결

- 파일: `problem_03_string_concat.c`
- 요구사항:
  - 두 문자열을 입력받는다.
  - 두 번째 문자열을 첫 번째 문자열 뒤에 이어 붙인다.
  - 문자 배열 기반 연결 함수를 구현한다.

### problem_04_string_compare - 문자열 비교

- 파일: `problem_04_string_compare.c`
- 요구사항:
  - 두 문자열을 입력받는다.
  - 문자 단위로 비교하는 함수를 구현한다.
  - 비교 결과를 정수 값으로 반환해 출력한다.

### problem_05_to_upper_string - 문자열 대문자 변환

- 파일: `problem_05_to_upper_string.c`
- 요구사항:
  - 문자열을 입력받는다.
  - 소문자를 대문자로 변환한다.
  - 변환된 문자열을 출력한다.

### problem_06_concat_five_lines - 여러 문자열 연결

- 파일: `problem_06_concat_five_lines.c`
- 요구사항:
  - 공백을 포함하는 문자열 5개를 입력받는다.
  - 문자열 연결 함수를 활용해 하나로 합친다.
  - 최종 연결 결과를 출력한다.

## 실행 방법

```bash
gcc problem_01_string_length.c -o problem_01_string_length
./problem_01_string_length
```
